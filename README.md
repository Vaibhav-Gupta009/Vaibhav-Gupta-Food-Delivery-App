# Vaibhav-Gupta-Food-Delivery-App
Web Dev assignment 5
This is a simple, responsive food‑delivery website that showcases a clear hero banner, food categories, featured restaurants, and a contact form. It uses basic HTML5 structure with sections for header, main content, and footer, and applies beginner‑friendly CSS Flexbox and Grid to arrange cards and forms so the layout adapts nicely from desktop to mobile screens.
